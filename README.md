## 1.説明
60秒間の中で敵を倒しながらスコアを獲得していくゲームを作成しました。<br>
作成理由としてはUnityと3Dゲームを作成する練習として作成しました。<br>
一定範囲内でランダムな座標にオブジェクトを生成したり、地形の高さにかかわらず地面にオブジェクトを生成したりなど、
これから3Dゲームを作成していく中で必要となりそうな技術を用いて作成しました。<br>

## 2.使用技術
C#,Unity

## 3.製作期間
2025年 9月、2026年5月<br>
2025年9月にUnityと3Dゲームの練習として作成し、その後2026年5月からブラッシュアップを行いました。<br>

## 4.アピールポイント、工夫点など
・アピールポイント①:強化エリアのランダム生成<br>
敵を倒すだけだとあまり面白くないと感じたので体力の回復、時間制限の追加、攻撃力アップの三つの効果をそれぞれ得ることができるエリアを一定時間ごとにランダム生成する機能を実装しました。 <br>
ランダムな位置に生成をする時、地形の高さに違いがあってもRaycastメソッドを用いて地面の高さを取得し、それに合わせて強化エリアを生成することでどんな地形でも地面の高さに合わせて強化エリアを生成できるように実装しました。<br>
<br>
・アピールポイント②:JSONファイルによるハイスコアの保存<br>
ゲームプレイの中で一番高かったスコアはJSONファイルに保存、ゲーム起動時にはJSONファイルから保存されているデータを読み込む機能を実装しました。これによってゲームを終了して再度プレイする際も、同じハイスコアでプレイできるようにしました。<br>
<br>
・アピールポイント③:プレイエリアの制限<br>
無意味にゲームプレイのエリアを広げると、プレイヤーが困惑すると感じたので Mesh Rendererを無効化したオブジェクトを設置することでプレイエリアを制限しました。<br>
<br>
・アピールポイント④:カメラ向きの制限<br>
カメラを上下に動かす時、制限がないとカメラの挙動がおかしくなってしまうので Clamp メソッドを用いて、カメラの上下の向きを制限しました。<br>
## 5.プレイしていただく時の注意点<br>
ゲーム内での実装が間に合いませんでしたので操作方法と簡単な説明を以下に記します。<br>
・移動: WASDキー<br>
・攻撃: スペースキー<br>
・カメラ操作: 矢印キー<br>
・ターゲットカメラ: 右シフトキー<br>
<br>
・緑エリア: 体力の回復<br>
・青エリア: 制限時間の増加<br>
・赤エリア: 攻撃力アップ<br>

## 6.プロジェクトについての注意点
1.プロジェクトをUnityで開く場合、「Title」がタイトル画面を、「Main」が実際にゲームを作成したsceneですのでプロジェクトを開いた時、sceneがこれらでなかったらAssetsフォルダの中にある「Main」もしくは「Title」に移動してください。
