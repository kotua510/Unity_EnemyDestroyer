## 1.説明
60秒間の中で敵を倒しながらスコアを獲得していくゲームを作成しました。
作成理由としてはUnityと3Dゲームを作成する練習として作成しました。
一定範囲内でランダムな座標にオブジェクトを生成したり、地形の高さにかかわらず地面にオブジェクトを生成したりなど、
これから3Dゲームを作成していく中で必要となりそうな技術を用いて作成しました。

## 2.使用技術
C#,Unity

## 3.製作期間
2025年 9月、2026年5月
2025年9月にUnityと3Dゲームの練習として作成し、その後2026年5月からブラッシュアップを行いました。

## 4.アピールポイント、工夫点など
・アピールポイント①:強化エリアのランダム生成<br>
文化祭での展示を想定し、短時間かつ直感的に遊べるよう設計しました。具体的には、ポーカーのルールを一部簡略化し、掛け金の設定を入力ではなくボタン操作にすることで、ポーカーが初めてのプレイヤーでも迷わずプレイできるようにしました。<br>
・アピールポイント②:当たり判定の厳密な実装<br>
本ゲームは1台の端末で複数人がプレイする仕様であるため、プレイヤー切り替え時に手札が見えてしまう課題がありました。これを防ぐため、プレイヤー切り替え時に確認画面を表示する仕組みを実装し、公平性を保ちました。<br>
・アピールポイント③:JSONファイルによるハイスコアの保存
・アピールポイント④:プレイエリアの制限
・アピールポイント⑤:カメラ向きの制限
## 5.プレイしていただく時の注意点
特にないです。<br>

## 6.プロジェクトについての注意点
1.プロジェクトをUnityで開く場合、「Title」がタイトル画面を、「Main」が実際にゲームを作成したsceneですのでプロジェクトを開いた時、sceneがこれらでなかったらAssetsフォルダの中にある「Main」もしくは「Title」に移動してください。
